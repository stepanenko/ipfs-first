### Fundamental building blocks of FP are:

- Immutable Data
- Pure Functions

### Pure Functions:

- Have input params
- No stateful values (not depend on variables outside)
- Return based on input
- No side effects

### Pure function are harder to write, but they are:

- Reusable
- Composable
- Easy to test
- Easy to cache
